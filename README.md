# Беговой дашборд

Статический дашборд для Google Sheets `Беговой_дневник_к_10км`.

## Что показывает

- план и факт за выбранный день;
- план и факт за календарную неделю;
- план и факт за месяц;
- список тренировок недели;
- отдельный блок для ОФП, мобилити и СБУ.

## Обновление данных

Страница пробует читать вкладку `Дневник` напрямую из Google Sheets. Для живого обновления на GitHub Pages таблица должна быть доступна публично хотя бы для просмотра по ссылке или опубликована в веб.

Если Google Sheets недоступен публично, дашборд всё равно открывается на сохранённом снимке данных от 28.06.2026.

## Деплой на GitHub Pages

1. Загрузить содержимое этой папки в репозиторий.
2. В настройках репозитория открыть `Settings -> Pages`.
3. Выбрать `Deploy from a branch`, ветку `main`, папку `/`.
4. Сохранить настройки и открыть выданную GitHub Pages ссылку.
